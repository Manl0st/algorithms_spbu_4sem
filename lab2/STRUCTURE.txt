ОСОБЕННОСТИ ИНТЕРФЕЙСА:

✓ Левая панель (параметры):
  - Выбор графа через радиокнопки
  - Выбор режима охлаждения (Classic/Cauchy)
  - Ввод параметров алгоритма
  - Поле ALPHA скрывается автоматически при выборе режима Cauchy
  - Кнопка "▶ Запустить" для начала вычисления

✓ Правая панель (результаты):
  - Текстовое поле со всеми параметрами и результатами
  - Процент улучшения по сравнению с начальным путем
  - Встроенные графики matplotlib (сходимость и охлаждение)
  - Динамическое обновление при каждом запуске

✓ Функциональность:
  - Валидация параметров перед запуском
  - Многопоточное выполнение (неблокирующий UI)
  - Встроенные графики без сохранения в файл
  - Поддержка 3 графов разных размеров
  - Оба режима охлаждения работают корректно

ТЕСТИРОВАНИЕ:

1. Built-in граф (6 вершин):
   - Classic режим: 44% улучшение за 167 шагов
   - Время: < 1 сек

2. BERLIN52 граф (52 города):
   - Classic режим: 67.3% улучшение за 137 шагов
   - Cauchy режим: 64.9% улучшение за 1124 шага
   - Время: 5-30 сек

3. WORLD666 граф (666 городов):
   - Поддерживается, требует больше времени
   - Рекомендуется использовать меньше ITERS_PER_T
