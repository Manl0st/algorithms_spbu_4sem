# Simulated Annealing TSP Solver

Простой и удобный интерфейс для решения задачи коммивояжера (TSP) методом имитации отжига.

## Возможности

- **2 режима охлаждения**:
  - Classic: T = T * ALPHA (более быстрое охлаждение)
  - Cauchy: T = T_START / (1 + step) (более медленное охлаждение)

- **3 графа для тестирования**:
  - Built-in: 6 вершин (для быстрого тестирования)
  - BERLIN52: 52 города (классический TSP benchmark)
  - WORLD666: 666 городов (большой граф)

- **Интерактивный ввод параметров**:
  - T_START: начальная температура
  - T_MIN: минимальная температура
  - ALPHA: коэффициент охлаждения (для режима Classic)
  - ITERS_PER_T: количество итераций при каждой температуре

- **Визуализация результатов**:
  - График сходимости (лучшая стоимость vs шаги)
  - График охлаждения температуры (температура vs шаги)
  - Статистика улучшения (в процентах)

## Использование

```bash
python3 annealing.py
```

Затем следуйте подсказкам интерфейса для выбора графа, режима и параметров.

## Архитектура кода

- `simulated_annealing()` - основной алгоритм SA
- `get_graph_choice()` - выбор графа
- `get_mode()` - выбор режима охлаждения  
- `get_parameters()` - ввод параметров
- `print_results()` - вывод результатов
- `plot_results()` - визуализация графиков

## Зависимости
