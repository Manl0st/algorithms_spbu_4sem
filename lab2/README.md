# TSP Solver - Simulated Annealing

Простое решение задачи коммивояжера (TSP) методом имитации отжига с графическим интерфейсом.

## Запуск

```bash
python3 annealing.py
```

## Интерфейс

**Левая панель:**
- Выбор графа (Built-in, BERLIN52, WORLD666)
- Выбор режима охлаждения (Classic, Cauchy)
- Ввод параметров (T_START, T_MIN, ALPHA, ITERS_PER_T)
- Кнопка запуска

**Правая панель:**
- Результаты работы
- Два графика (сходимость и температура)

## Параметры

- **T_START** - начальная температура
- **T_MIN** - минимальная температура
- **ALPHA** - коэффициент охлаждения (для режима Classic)
- **ITERS_PER_T** - итераций при каждой температуре

## Графы

- Built-in: 6 вершин
- BERLIN52: 52 города
- WORLD666: 666 городов

## Режимы охлаждения

- **Classic:** T = T × ALPHA
- **Cauchy:** T = T_START / (1 + step)
