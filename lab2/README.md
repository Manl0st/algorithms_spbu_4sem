# TSP Solver - Simulated Annealing

Решение задачи коммивояжера методом имитации отжига с простым графическим интерфейсом.

## Запуск

```bash
python3 annealing.py
```

## Интерфейс

- **Левая панель** - параметры и результаты
  - Выбор графа (радиокнопки)
  - Выбор режима охлаждения (выпадающий список)
  - Ввод параметров (T_START, T_MIN, ALPHA, ITERS_PER_T)
  - Вывод результатов (длина пути, маршрут, время)

- **Правая панель** - графики (вертикально)
  - График сходимости алгоритма
  - График охлаждения температуры

## Графы

- **Built-in:** 6 вершин
- **BERLIN52:** 52 города
- **WORLD666:** 666 городов

## Режимы охлаждения

- **Classic:** T = T × ALPHA
- **Cauchy:** T = T_START / (1 + step)

## Параметры

- **T_START** - начальная температура (по умолчанию 100.0)
- **T_MIN** - минимальная температура (по умолчанию 0.089)
- **ALPHA** - коэффициент охлаждения (по умолчанию 0.95)
- **ITERS_PER_T** - итераций при каждой температуре (по умолчанию 30)
