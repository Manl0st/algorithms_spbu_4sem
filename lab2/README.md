# Simulated Annealing TSP Solver

Полнофункциональное приложение для решения задачи коммивояжера (TSP) методом имитации отжига с графическим интерфейсом.

## Возможности

- **2 режима охлаждения**:
  - Classic: T = T × ALPHA (более быстрое охлаждение)
  - Cauchy: T = T_START / (1 + step) (более медленное охлаждение)

- **3 графа для тестирования**:
  - Built-in: 6 вершин (для быстрого тестирования)
  - BERLIN52: 52 города (классический TSP benchmark)
  - WORLD666: 666 городов (большой граф)

- **Графический интерфейс (GUI)**:
  - Выбор графа через радиокнопки
  - Выбор режима охлаждения
  - Ввод параметров: T_START, T_MIN, ALPHA (для Classic), ITERS_PER_T
  - Динамическое отображение/скрытие параметра ALPHA
  - Вывод результатов в реальном времени
  - Встроенные графики matplotlib (сходимость и охлаждение)

- **Визуализация результатов**:
  - График сходимости (лучшая стоимость vs шаги)
  - График охлаждения температуры (температура vs шаги)
  - Статистика улучшения (в процентах)

## Запуск

### GUI версия (рекомендуется)

```bash
python3 gui.py
```

Откроется окно с интерфейсом, где можно:
1. Выбрать граф
2. Выбрать режим охлаждения
3. Ввести параметры алгоритма
4. Нажать "▶ Запустить"
5. Просмотреть результаты и графики

### Консольная версия

```bash
python3 annealing.py
```

## Архитектура кода

### Файлы
- `gui.py` - Графический интерфейс с tkinter и встроенными графиками
- `annealing.py` - Консольная версия с интерактивным вводом
- `matrices.py` - Предварительно сгенерированные матрицы графов
- `gen_matrices.py` - Генератор матриц из STP файлов

### Классы и функции
- `TSPSolver` - класс с методами алгоритма SA
  - `solve()` - основной алгоритм
  - `route_cost()` - расчет стоимости пути
  - `random_path()` - генерация случайного пути
  - `mutate_path()` - мутация пути
  - `next_temperature()` - обновление температуры

- `TSPGUI` - класс графического интерфейса
  - `setup_ui()` - создание элементов интерфейса
  - `validate_parameters()` - проверка корректности параметров
  - `run_algorithm()` - запуск алгоритма
  - `display_results()` - отображение результатов
  - `plot_results()` - построение графиков

## Зависимости

- `matplotlib` - для построения графиков
- `tkinter` - для графического интерфейса (обычно встроен в Python)
